# dsc

Official CLI for DSC-PUCP.

## Installation

### Quick install (macOS / Linux)

```bash
curl -fsSL https://raw.githubusercontent.com/DSC-PUCP/dsc-cli/main/install.sh | sh
```

### Homebrew (macOS)

```bash
brew install DSC-PUCP/tap/dsc
```

### From source

```bash
git clone https://github.com/DSC-PUCP/dsc-cli.git
cd dsc-cli
make install
```

## Setup

```bash
dsc auth login
dsc config set gemini-key
```
